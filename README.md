Technical Assessment for GT MOTIVE  
Microservice with NET 8 AND PostgreSQL DB with Hexagonal Architecture  
This app is dockerized  
To run in container:    
1. Go to the microservicio_GtMotive Folder (Make sure you are in the folder where the docker-compose file is located.) and execute --> docker compose up --build    
2. Open de Browser and go to : http://localhost:5069/swagger/index.html
